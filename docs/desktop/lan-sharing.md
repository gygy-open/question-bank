# 局域网共享

局域网共享是桌面版的一个**可选开关**：开启后，你这台电脑上的题库会开放给**同一局域网**内的同事访问，大家共用同一份数据。适合教研组、机房等在同一网络内的小团队轻量协作。

::: warning 默认关闭 · 仅限可信内网
局域网共享**默认关闭**。它只应在你信任的内网（如办公室 / 机房）中开启，**不要**直接暴露到公网。若需要长期、面向团队的在线服务，请使用 [服务器版](/server/docker)。
:::

## 工作原理

- **关闭时**：内置服务绑定 `127.0.0.1`，仅本机可访问。
- **开启时**：内置服务改为绑定 `0.0.0.0`，并添加一条 Windows 防火墙放行规则，同网段设备可通过 `http://<你的IP>:<端口>` 访问。

数据始终保存在**你这台电脑**的 SQLite 中，其他人是通过网络访问你的服务，因此：**你的电脑必须开机并运行应用，他人才能访问。**

## 开启步骤

1. 右键托盘图标，勾选 **「局域网共享」**。
2. 系统会弹出 **UAC 提权确认**（用于添加防火墙规则），点击「是」。
3. 服务会自动重启并绑定到局域网。

## 获取访问地址

开启后，同事需要用**你电脑的局域网 IP** 访问。查看本机 IP:

打开 PowerShell 或命令提示符，执行：

```powershell
ipconfig
```

在结果中找到当前网络适配器的 **IPv4 地址**（通常形如 `192.168.x.x`）。

同事在浏览器访问：

```
http://192.168.x.x:<端口>
```

## 多人使用注意事项

- **账号**：每个人用各自的账号登录。管理员可在 [用户与权限](/admin/users) 中为同事创建账号。
- **并发**：SQLite 适合小规模并发；人数较多或写入频繁时，建议迁移到 [服务器版](/server/docker)。
- **主机常开**：共享依赖你这台「主机」电脑；它关机或退出应用后，他人无法访问。
- **IP 变化**：若主机 IP 由 DHCP 分配，重启后地址可能变化，可请网管为主机绑定固定 IP。

## 关闭共享

右键托盘图标，取消勾选 **「局域网共享」**。服务会重启并重新绑定回 `127.0.0.1`，防火墙规则在卸载时清理。

## 安全建议

- 仅在可信内网开启，切勿映射到公网。
- 为每位同事分配独立账号，避免共用。
- 使用强密码，定期在 [操作审计](/features/activity-logs) 查看操作记录。
- 如需公网访问 / 更强安全性，请部署 [服务器版](/server/docker)。
