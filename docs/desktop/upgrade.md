# 升级与卸载

## 升级到新版本

1. 从 [GitHub Releases](https://github.com/gygy-open/question-bank/releases) 下载最新安装包。
2. 右键托盘图标 → 「退出」，关闭当前应用。
3. 运行新安装包，**覆盖安装**即可。

::: tip 数据会保留
升级采用覆盖安装，程序文件被替换，但你的数据目录 `%APPDATA%\QuestionBank`（SQLite 数据库、上传文件、配置）**不受影响**。为稳妥起见，升级前仍建议先[备份](/desktop/personal#备份与恢复)。
:::

安装程序在升级时会自动结束正在运行的旧版本进程，如遇「文件被占用」提示，请先手动退出应用再重试。

## 卸载

1. 右键托盘图标 → 「退出」。
2. 在 Windows「设置 → 应用」或「控制面板 → 程序和功能」中找到 **Question Bank** 并卸载。

卸载程序会：

- 结束正在运行的应用进程；
- 移除程序文件与开机自启项；
- 清理局域网共享添加的防火墙规则。

## 卸载后是否保留数据？

卸载**不会**自动删除你的数据目录 `%APPDATA%\QuestionBank`。

- 想彻底清除：手动删除该目录。
- 想保留 / 迁移：先复制该目录再删除；重新安装后放回即可恢复。

::: warning 删除不可恢复
手动删除 `%APPDATA%\QuestionBank` 会永久删除你的全部题库数据，操作前请确认已备份。
:::
