# 个人使用

个人使用是桌面版的默认模式：内置服务只绑定本机 `127.0.0.1`，**仅你这台电脑能访问**，数据保存在本地 SQLite。适合个人备课、录题、组卷。

## 日常流程

1. **启动**：开机后（若勾选了开机自启）应用自动在后台运行；否则从开始菜单启动。
2. **打开题库**：右键托盘图标 → 「打开题库」，浏览器打开界面并登录。
3. **配置 AI**（首次）：进入设置，添加 AI 供应商与模型，详见 [AI 供应商与模型](/admin/ai-config)。
4. **录入题目**：
   - 手动新建，或
   - 用 [智能导入](/features/import) 上传 Word / Markdown / 图片，由 AI 抽取。
5. **管理与组卷**：在 [题库管理](/features/questions) 中筛选、编辑、审核，用 [组卷 / 试题篮](/features/papers) 导出试卷。

## 数据存放

个人使用模式下，所有数据都在本机：

```
%APPDATA%\QuestionBank
```

包含 SQLite 数据库、上传的原始文件、配置与日志。没有云端同步，数据完全由你掌控。

## 备份与恢复

**备份：**

1. 右键托盘图标 → 「退出」，确保应用已关闭。
2. 复制整个 `%APPDATA%\QuestionBank` 目录到安全位置（移动硬盘 / 网盘）。

**恢复 / 迁移到新电脑：**

1. 在新电脑上安装桌面版并至少启动一次（生成目录）。
2. 退出应用。
3. 用备份覆盖新电脑的 `%APPDATA%\QuestionBank` 目录。
4. 重新启动应用。

::: warning 注意
备份 / 恢复务必在应用**已退出**的状态下进行，避免 SQLite 数据库正在写入导致文件损坏。
:::

## 从个人使用切换到多人协作

如果后续需要让同事一起用，有两个选择：

- **同网段临时协作** → 打开 [局域网共享](/desktop/lan-sharing) 开关即可，无需迁移数据。
- **正式团队部署** → 迁移到 [服务器版](/server/docker)。
