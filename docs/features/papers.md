# 组卷 / 试题篮

试题篮用于临时收藏题目，组卷则把选中的题目导出为试卷文件（Word / LaTeX）。

## 试题篮

- 在 [题库管理](/features/questions) 浏览时，把中意的题目**加入试题篮**。
- 试题篮保存在浏览器本地（localStorage），可随时增删。
- 用于跨多次筛选、逐步攒题，再统一组卷。

::: warning 非持久化
试题篮存在浏览器本地，清缓存或换设备会丢失。请在攒齐后及时组卷导出。
:::

## 组卷导出

选定题目后，可导出为试卷文件。

**导出格式：**

- **Word(DOCX)** — 默认
- **LaTeX** — LaTeX 源码（打包为 zip）

**内容选项（可勾选包含）：**

| 选项 | 说明 |
|------|------|
| `include_answer` | 参考答案 |
| `include_analysis` | 解析 |
| `include_explanation` | 说明 / 解释 |
| `include_summary` | 小结 |
| `include_source` | 来源信息 |

导出时可设置试卷**标题**，系统生成文件供下载。

## 建议流程

1. 在题库中按学科 / 知识点 / 难度筛选。
2. 逐题加入试题篮。
3. 打开试题篮，设置标题与内容选项。
4. 选择格式导出并下载。
