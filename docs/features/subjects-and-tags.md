# 学科与标签

学科与标签用于组织题库的分类维度。

## 学科（科目）

学科是题库的顶层分类，题目、知识点均归属到学科。

字段：名称、标识（slug，唯一）、描述、审核阈值 `required_review_count`（默认 1）。

操作：新建、编辑、删除学科；每个学科可维护自己的 [知识点体系](/features/knowledge-points)。

## 标签

标签是灵活的分类维度，与题目**多对多**关联。每个标签有名称、所属**分类**与颜色。

**内置标签分类：**

| 分类 | 标识 | 示例 |
|------|------|------|
| 年份 | `year` | 2023、2024 |
| 来源 | `source` | 某某模拟卷 |
| 年级 | `grade` | 高一、高二 |
| 学期 | `semester` | 上学期、下学期 |
| 考试类型 | `exam_type` | 期中、期末、模拟 |
| 特征 | `feature` | 典型、压轴、同步 |
| 通用 | `general` | 默认 |

操作：新建（自动查重）、编辑、删除（级联清理关联）。

## 标签分类（管理）

标签分类定义可用的标签分组，含名称、slug（唯一）、排序、启用状态。

::: tip 权限
**标签分类的管理为超级管理员专属**。普通用户可使用已有分类下的标签，但不能增删分类本身。
:::

删除某个标签分类会级联删除其下标签，请谨慎操作。
