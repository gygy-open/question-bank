# 文件预览

文件预览让你在浏览器中直接查看 DOCX 与 Markdown 原始文件，无需下载。

## 支持格式

| 格式 | 渲染方式 |
|------|---------|
| DOCX | 使用 `docx-preview` 在浏览器内渲染 |
| Markdown | 内置 Markdown 预览组件渲染 |

## 使用

预览页通过文件地址加载并按扩展名判断类型，直接在页面内呈现，并提供下载入口。常见于查看智能导入的原始素材。

::: tip
预览由前端直接读取文件地址渲染，原始文件存放于上传目录（服务器版为 `backend/uploads`，桌面版在数据目录内）。
:::
