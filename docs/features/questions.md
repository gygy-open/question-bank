# 题库管理

题库管理提供题目的检索、编辑、批量操作、母子题结构与软删除。

## 题型

| 题型 | 标识 | 说明 |
|------|------|------|
| 单选 | `single_choice` | 单个正确选项 |
| 多选 | `multiple_choice` | 多个正确选项 |
| 判断 | `true_false` | 对 / 错 |
| 填空 | `fill_in_the_blank` | 支持**多个空**且每空**多解** |
| 解答 | `free_response` | 主观 / 简答题 |

题干与解析支持富文本与 **LaTeX 公式**。

## 筛选

题库列表支持多条件组合筛选：

- 学科、知识点、标签（可多选）
- 题型、难度、状态
- 导入任务（按批次 / 批次名）
- 关键词（题干全文搜索）
- 创建人、审核人、审核次数
- 来源、是否仅顶层题（母题）

## 题目字段

- `content` 题干（Markdown / HTML）
- `options` 选项（选择题）：`[{"label":"A","content":"..."}, ...]`
- `answer` 参考答案（填空题为多解结构，如 `[["a","A"],["b","B"]]`）
- `thinking` / `analysis` / `summary` 思路 / 解析 / 小结
- `difficulty` 难度（1–5，默认 1）
- `status` 状态（见 [审核工作流](/features/review-workflow)）

## 批量操作

- **批量创建**：一次写入多题（支持母子题嵌套），智能导入即基于此。
- **批量删除**：软删除，记录操作日志。
- **批量更新**：批量修改来源等元数据。

## 母子题（母子题结构）

- 母题 `parent_id` 为空，可包含多个子题。
- 子题通过 `parent_id` 关联到母题。
- 删除母题会级联删除其子题。
- 适用场景：一拖多的材料题、变式题、子问。

## 软删除

- 删除题目只是打上 `deleted_at` 时间戳，默认查询会自动排除。
- 保留审计痕迹，支持恢复。

::: warning 与彻底删除区别
软删除可恢复；如需彻底清除请由管理员通过运维手段处理。
:::

## 关联

题目可关联：标签（多对多）、知识点（多对多）、导入任务、创建 / 更新人（审计）、审核日志。
