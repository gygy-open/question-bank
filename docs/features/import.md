# 智能导入

智能导入让你上传原始素材，由 AI 自动抽取成结构化题目，遵循 **上传 → 审核 → 入库** 三步流程。

## 支持格式

| 类型 | 扩展名 | 说明 |
|------|--------|------|
| Word | `.docx` | 解析文档文本 |
| Markdown | `.md` / 粘贴文本 | 直接解析 Markdown |
| 图片 | `.png` `.jpg` `.jpeg` `.webp` | 通过视觉模型 OCR 识别 |

::: tip 前置条件
导入依赖 AI 模型：文本类使用文本模型，图片类使用**视觉模型**。请先在 [AI 供应商与模型](/admin/ai-config) 配置并激活对应模型。
:::

## 三步流程

### 1. 上传

上传一个或多个文件，或直接粘贴 Markdown / 图片内容。可选择模式：

- **抽取（extract）**：默认，从素材中抽取已有题目。
- **解答（solve）**：让 AI 对题目进行解答。

### 2. 审核

AI 抽取后，逐题展示可编辑的结果，字段包括：

- 题干内容、题型、选项、答案、解析、思路、难度
- AI 建议的标签（年份、来源、年级、学期、考试类型、特征等）

你可以逐题修改、删除、调整后再确认。

### 3. 入库

确认后，题目批量写入题库，并与本次**导入任务**关联，便于后续按批次筛选和管理。

## 导入任务状态

每次导入会生成一个导入任务，状态包括：

| 状态 | 含义 |
|------|------|
| `PENDING` | 排队中 |
| `PROCESSING` | 处理中 |
| `COMPLETED` | 已完成 |
| `FAILED` | 失败 |
| `CANCELLED` | 已取消 |

## 批次管理

- **队列状态**：查看当前排队 / 处理中 / 完成 / 失败 / 取消的统计与活动任务。
- **导入历史**：分页查看历次导入，可按状态、用户筛选。
- **权限**：超级管理员可见全部导入任务；普通用户仅见自己的。

## 常见问题

- **抽取结果不准？** 调整 [Prompt 模板](/admin/prompts) 或更换更强的模型。
- **图片识别失败？** 确认已激活**视觉模型**，且图片清晰。
- **卡在处理中？**（服务器版）检查 [Worker 日志](/server/operations#后台任务-worker)。
