Question Bank —— 桌面版安装说明(Windows)
================================================

安装(最简单)
--------------
1. 双击 QuestionBank-Setup-x.y.z.exe
2. 弹出“是否允许更改”时点“是”
3. 按向导点“下一步 → 安装 → 完成”

安装程序会:装到 Program Files、设置开机自启(登录后自动显示托盘图标)、
在开始菜单创建“Question Bank / 打开题库 / 卸载题库”快捷方式。全程无需命令行。

托盘图标(右下角通知区)
------------------------
安装完成后右下角会出现题库图标,右键菜单:
  打开题库      在浏览器打开 http://localhost:8000
  局域网共享    开/关(默认关)。开启后其他终端可访问,见下方。
  重启          重启后台服务
  退出          关闭题库(下次开机或手动启动会再次运行)

首次使用
--------
在本机浏览器打开 http://localhost:8000 完成安装向导
(选择数据库、创建管理员账号)。也可用托盘“打开题库”。

让其他终端访问(局域网共享)
----------------------------
1. 右键托盘图标 → 勾选“局域网共享”
2. 弹出“是否允许更改”时点“是”(用于放行防火墙端口)
3. 其他终端浏览器打开 http://<本机IP>:8000
   查看本机 IP:命令提示符运行 ipconfig,找 IPv4 地址。
不需要共享时取消勾选即可,防火墙端口会自动关闭。

卸载
----
Windows“设置 → 应用”里找到 Question Bank 卸载,或用开始菜单“卸载题库”。
卸载会关闭托盘、移除开机自启和防火墙规则;应用数据不会被删除。

数据位置
--------
应用数据(数据库、config.json、上传文件)在 %APPDATA%\QuestionBank,
升级不会丢数据。备份/迁移直接复制该目录即可。
